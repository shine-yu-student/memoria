## JSON 序列化

### 单资源导出

每篇文章/词书/句书都可单独导出为 JSON：

```json
{
  "id": "abc123...",
  "title": "文章标题",
  "content": "全文内容……",
  "sentences": ["句子1", "句子2"],
  "createdAt": 1700000000000
}
```

通过 `SingleExportBtn` 组件触发下载：

```tsx
<SingleExportBtn label="文章名" data={article} filename="article-标题.json" />
```

### 单资源导入

通过 `<input type="file">` 选择 JSON 文件：

```tsx
<SingleImportBtn label="文章 (JSON)" onData={handleImportArticle} />
```

导入时校验数据格式，自动补全缺失字段（如使用 `uid()` 生成新 ID）。

### 全量备份格式

```typescript
interface MemoriaData {
  version: 1;
  articles: Article[];
  wordBooks: WordBook[];
  sentenceBooks: SentenceBook[];
  exportedAt: number;
}
```

全量导入会**覆盖**当前所有数据：

```typescript
function importAll(data: MemoriaData): void {
  saveArticles(data.articles || []);
  saveWordBooks(data.wordBooks || []);
  saveSentenceBooks(data.sentenceBooks || []);
}
```

## 文件处理

### 下载

使用 `Blob + URL.createObjectURL` 实现浏览器端文件下载：

```typescript
function downloadJSON(data: unknown, filename: string) {
  const blob = new Blob([JSON.stringify(data, null, 2)], { type: 'application/json' });
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = filename;
  a.click();
  URL.revokeObjectURL(url);
}
```

### 上传

使用 `FileReader` 读取文件内容并解析 JSON：

```typescript
function readJSONFile(file: File): Promise<unknown> {
  return new Promise((resolve, reject) => {
    const reader = new FileReader();
    reader.onload = () => resolve(JSON.parse(reader.result as string));
    reader.onerror = () => reject(new Error('文件读取失败'));
    reader.readAsText(file);
  });
}
```

## 错误处理

- JSON 解析失败 → 弹出错误提示
- 无效的数据格式 → 格式校验后拒绝导入
- 文件读取失败 → 捕获异常并提示
