# 01 — 项目概览

## 项目简介

**Memoria** 是一个基于 TypeScript + React 的本地化记忆助手应用。它帮助用户通过"主动回忆"的方法来记忆中文文章和英文词汇/句子。

## 功能总览

### 语文记忆模块

- **文章管理**：支持导入多篇中文文章，自动按分隔符切分为句子
- **全篇记忆**：用户凭记忆输入整篇文章，系统与原文逐字符比对并标记差异
- **部分记忆**：
  - **随机记忆**：系统随机选取句子作为填空题
  - **指定记忆**：用户自行勾选需要记忆的句子
- **结果反馈**：以绿色（正确）、橙色（遗漏）、红色删除线（多余/错误）显示差异

### 英语记忆模块

- **词书管理**：支持创建和管理英语单词/短语-中文对应关系的词书
- **句书管理**：支持创建和管理英语句子-中文翻译的句书
- **灵活选择**：支持选择整本书或书中的部分条目进行记忆
- **混合记忆**：可同时从多本书中选择条目进行记忆
- **翻译提示**：展示中文意义，用户输入对应的英文内容

### 导入导出系统

- **单资源导出**：每篇文章/词书/句书可单独导出为 JSON 文件
- **单资源导入**：从 JSON 文件导入单篇文章/词书/句书
- **全量导出**：将所有数据导出为单个 JSON 备份文件
- **全量导入**：从备份文件恢复全部数据

## 技术栈

| 技术 | 用途 |
|------|------|
| TypeScript 5.6+ | 类型安全 |
| React 18.3+ | UI 框架 |
| Vite 6 | 构建工具 |
| diff 库 | 文本差异比较 |
| localStorage | 客户端持久化 |

## 运行方式

```bash
npm install
npm run dev      # 开发模式
npm run build    # 生产构建
npm run preview  # 预览构建结果
```
