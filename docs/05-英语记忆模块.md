## 内容选择机制

新版的选择机制已整合到 BookManager 列表页面中，不再使用独立的 selection 页面。

### 选择模式

在英语记忆列表页，点击标题旁的"🧠 开始英语记忆"按钮进入选择模式：

```
┌──────────────────────────────────────────────────────────────┐
│  选择要记忆的内容                  [✕ 退出选择] [开始记忆 (N)]│
│  已选 N 项。点击条目选择，按住 Shift 可范围选择。              │
├──────────────────────────────────────────────────────────────┤
│  📗 词书名（N 词）                          [🔽 展开]        │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  ☐ word1 — 意思1                                     │  │
│  │  ☑ word2 — 意思2  ← 已选中                           │  │
│  │  ...                                                 │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
│  鼠标悬停时右下角显示复选框：                                  │
│  📗 词书名（N 词）                          [🔽 展开]  [☐]  │
└──────────────────────────────────────────────────────────────┘
```

- **选择/取消选择**：点击"开始记忆"进入选择模式，"开始记忆"变为"开始记忆(N)"。
- **退出选择**：点击"✕ 退出选择"恢复初始状态。
- **整书选择**：悬停词书时右下角显示复选框，点击选中/取消整本书。
- **部分选择**：展开词书后可单独点击条目选择，条目行高亮显示选中状态。
- **范围选择**：按住 Shift 点击两个条目，选中其间所有条目。
- **混合选择**：可同时从多个词书和句书中选择条目。

## 闪卡记忆系统

### 核心算法

```
初始状态：
  queue      = 打乱后的全部待测词条
  retestQueue = []    （答错后待复习的词条）
  answeredCount = 0   （已答题数，用于计算复习时机）

取下一个词（nextWord）：
  1. answeredCount++
  2. 检查 retestQueue 是否有到期的（dueAfter ≤ answeredCount）
     → 有：取出作为当前词
  3. 否则从 queue 取下一个
  4. queue 空但 retestQueue 还有 → 强制清出
  5. 两者皆空 → 完成
```

### 答题界面

- **顶部进度条**：显示进度百分比、已完成/总数、错误数。
- **计时**：右上角显示实时用时 `MM:SS`。
- **中文提示**：卡片中央显示中文含义。
- **输入框**：用户输入对应的英文。
- **答错处理**：展示正确答案，要求重新输入，正确后送入 retest 队列。

### 答题状态机

```
                  ┌─────────────┐
                  │   showing   │  ← 显示中文，等待输入
                  └──────┬──────┘
                   正确 │    │ 错误
                  ┌─────▼┐   │
                  │ 完成 │   │
                  └──────┘   │
                             ▼
                  ┌─────────────┐
                  │ wrong-reveal│  ← 显示正确答案，要求重新输入
                  └──────┬──────┘
                   输对了 │
                  ┌─────▼──────────────┐
                  │ 送入 retestQueue   │
                  │ dueAfter = 当前计数 │
                  │           + 3~5   │
                  └────────────────────┘
```

### 结果页面

- **总题数**：本次记忆的总词条数。
- **用时**：记忆总耗时（原"答对次数"已替换为"用时"）。
- **曾答错**：错误次数统计。
- **错词列表**：可展开查看所有曾答错的单词及错误次数。

### 比对规则

```typescript
userInput.trim().toLowerCase() === correct.trim().toLowerCase()
```

- 忽略首尾空白。
- 忽略大小写差异。
