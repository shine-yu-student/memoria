# 03 — 数据类型

所有类型定义集中在 `src/types/index.ts`。以下是核心数据模型。

## Article（文章）

```typescript
interface Article {
  id: string;           // 唯一标识
  title: string;        // 文章标题
  content: string;      // 全文原文
  sentences: string[];  // 按分隔符切分后的句子列表
  createdAt: number;    // 创建时间戳
}
```

**说明**：`sentences` 字段在导入时由 `splitIntoSentences()` 函数自动生成，以中文标点（。！？，、；：……—）和换行符为分隔。

## WordBook（词书）

```typescript
interface WordBook {
  id: string;           // 唯一标识
  title: string;        // 词书名称
  entries: WordEntry[]; // 词条列表
  createdAt: number;
}

interface WordEntry {
  id: string;       // 唯一标识
  english: string;  // 英语单词/短语
  chinese: string;  // 中文释义
}
```

## SentenceBook（句书）

```typescript
interface SentenceBook {
  id: string;           // 唯一标识
  title: string;        // 句书名称
  entries: SentenceEntry[]; // 句条列表
  createdAt: number;
}

interface SentenceEntry {
  id: string;       // 唯一标识
  english: string;  // 英语句子
  chinese: string;  // 中文翻译
}
```

## BlankItem（填空题项）

```typescript
interface BlankItem {
  id: string;          // 唯一标识
  correct: string;     // 正确原文
  userInput: string;   // 用户输入
  graded: boolean;     // 是否已批改
  correctFlag: boolean; // 是否完全正确
}
```

## MemoriaData（全量备份）

```typescript
interface MemoriaData {
  version: number;           // 格式版本号
  articles: Article[];       // 所有文章
  wordBooks: WordBook[];     // 所有词书
  sentenceBooks: SentenceBook[]; // 所有句书
  exportedAt: number;        // 导出时间戳
}
```

## session 类型（记忆会话）

```typescript
interface ChineseMemorySession {
  articleId: string;
  articleTitle: string;
  mode: 'full' | 'partial-random' | 'partial-custom';
  blankIndices: number[];
  blanks: BlankItem[];
  completed: boolean;
  score: number;
}

interface EnglishMemorySession {
  bookIds: string[];
  mode: string;
  blanks: BlankItem[];
  completed: boolean;
  score: number;
}
```
