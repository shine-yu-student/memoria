## 文章导入流程

1. 用户在文本框中输入或粘贴文章正文（可选填写标题）
2. 点击「导入文章」按钮
3. 系统调用 `splitIntoSentences(content)` 自动切分句子
4. 生成 `Article` 对象并存入 localStorage

## 句子切分算法

`src/utils/splitter.ts`：

```typescript
const DELIMITER_RE = /[。！？，、；：……—\n\r\u3000\s]+/;

function splitIntoSentences(text: string): string[] {
  return text
    .split(DELIMITER_RE)
    .map(s => s.trim())
    .filter(s => s.length > 0);
}
```

- 以中英文标点（句号、感叹号、问号、逗号、顿号、分号、冒号、省略号、破折号）以及空白字符和换行为分割
- 过滤空字符串

## 记忆流程

### 全篇记忆

1. 用户点击「全篇记忆」
2. 显示一个空白文本框
3. 用户凭记忆输入整篇文章
4. 点击「提交检查」
5. 系统对全文进行字符级 diff 比对
6. 显示评分和每处差异

### 部分记忆 — 随机模式

1. 系统调用 `pickRandomIndices()` 选取约 40% 的句子
2. 进入答题界面后，**全文在可滚动容器中完整显示**
3. 被选中的句子位置显示为**虚线绿色输入框**，未被选中的句子保持原文显示
4. 用户在全文中定位输入框，逐句填写
5. 提交后逐句比对

### 部分记忆 — 指定模式

1. 页面展示全部句子的复选框列表
2. 用户自行勾选需要记忆的句子
3. 点击「开始记忆」
4. 后续流程同随机模式

## 部分记忆 — 答题界面

进入答题后，页面展示文章的**完整全文**（可滚动），被选中的句子替换为输入框：

```
原文：句子1。句子2。句子3。句子4。
选择：句子2、句子4

显示效果：
  句子1。[ 输入框  ]。句子3。[ 输入框  ]。
```

- **绿色虚线输入框**：醒目提示需要填写的空白
- **分隔符自动保留**：逗号、句号等标点保持在输入框之间
- **可滚动容器**：长文章也可在固定区域内上下滚动查看
- 提交后比对时，使用字符级 diff 逐句检查
