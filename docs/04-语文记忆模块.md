# 04 — 文章记忆模块

> 原"语文记忆模块"，现更名为"文章记忆模块"以提高辨识度。

## 文件说明

| 文件 | 职责 |
|------|------|
| `src/components/chinese/ArticleManager.tsx` | 文章列表管理、卡片展示、搜索、新增文章 |
| `src/components/chinese/ArticleDetailView.tsx` | 文章编辑、标题内联修改、导出、开始记忆 |
| `src/components/chinese/ArticleMemoryView.tsx` | 记忆模式选择、全篇/部分记忆答题交互 |
| `src/utils/splitter.ts` | 中文句子切分 |

## 文章管理页面

### 页面布局

```
┌──────────────────────────────────────────────────────────────┐
│  📖 文章记忆  [✏️ 创建文章]                    [搜索框]      │
│──────────────────────────────────────────────────────────────│
│  📚 我的文章                                                   │
│  ┌────────────────────────────────────────────────────────┐   │
│  │ [文章卡片1] [文章卡片2] [文章卡片3]                     │   │
│  │ [文章卡片4]                                            │   │
│  └────────────────────────────────────────────────────────┘   │
└──────────────────────────────────────────────────────────────┘
```

- **标题**："📖 文章记忆"位于左上角。
- **创建按钮**："✏️ 创建文章"位于标题右侧。
- **搜索框**：位于行最右端，支持按标题搜索。
- **分隔线**：标题行下方的分隔线。
- **我的文章**：图标 + 文字。
- **文章容器**：圆角矩形，内部以 flex-wrap 排列文章卡片。
- **文章卡片**：小圆角矩形，显示标题。鼠标悬停时右下角显示删除按钮。

### 交互流程

1. 点击"创建文章" → 弹出输入框 → 确认后自动跳转到编辑页面。
2. 点击文章卡片（非删除按钮区域）→ 进入编辑页面。
3. 悬停文章卡片 → 右下角显示🗑️删除按钮。
4. 点击删除按钮 → 确认弹窗 → 确认后删除。

## 文章编辑页面

### 页面布局

```
┌──────────────────────────────────────────────────────────────┐
│  ← 返回   文章标题 (点击修改)          [⬇️ 导出] [🧠 开始记忆] │
│──────────────────────────────────────────────────────────────│
│  ┌────────────────────────────────────────────────────────┐  │
│  │  （文本编辑区域，占剩余高度）                            │  │
│  │  用户可直接在此编辑文章内容                              │  │
│  │                                                        │  │
│  │                                  共 N 句 / M 字        │  │
│  └────────────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────────────┘
```

- **返回按钮**：左上角"← 返回"。
- **文章标题**：大号字体，点击后变为输入框，可内联修改标题。按 Enter 保存，按 Escape 取消，失焦时自动保存。
- **操作按钮**：标题右侧放置导出和开始记忆按钮。
- **分隔线**：标题与编辑区域之间的分隔线。
- **编辑区域**：圆角矩形，充满剩余空间。用户可在textarea中直接编辑文章内容，系统自动切分句子。

## 记忆模式选择

点击"开始记忆"后进入模式选择页面，以卡片形式显示：

```
┌─────────────────────────────────────────────┐
│  📝 全篇记忆                    [选择]       │
│  不提供任何提示，凭记忆输入整篇文章。          │
├─────────────────────────────────────────────┤
│  📄 部分记忆                    [选择]       │
│  只记忆文章中部分句子。                      │
└─────────────────────────────────────────────┘
```

## 全篇记忆

1. 用户点击「全篇记忆」→ 显示空白文本框。
2. 用户凭记忆输入整篇文章。
3. 点击「提交检查」→ 进入结果页面。
4. 系统对全文进行字符级 diff 比对，显示评分和每处差异。

## 部分记忆

### 随机模式

```
┌──────────────────────────────────────────────────────────────┐
│  [🎲 随机记忆]   选取比例：[0.4] (共 N 句，约 M 句)           │
│──────────────────────────────────────────────────────────────│
│  ✋ 指定记忆                                                   │
│  ┌────────────────────────────────────────────────────────┐  │
│  │ [句子1] [句子2] [句子3] [句子4] [句子5]                 │  │
│  │ [句子6] ...                                           │  │
│  └────────────────────────────────────────────────────────┘  │
│              [开始记忆选中的 K 句]                            │
└──────────────────────────────────────────────────────────────┘
```

1. **随机记忆**：上方显示"🎲 随机记忆"按钮 + 比例输入。点击后随机选取对应比例的句子。
2. **指定记忆**：下方分隔线 + "✋ 指定记忆"标题。文章句子以可点击按钮形式显示在圆角矩形中。
3. **句子选择**：点击按钮切换选中状态。支持 Shift+点击范围选择（按住 Shift 点击两个句子，选中其间所有句子）。
4. 点击"开始记忆选中的 N 句"进入答题界面。

### 答题界面

进入答题后，页面展示文章的完整全文（可滚动），被选中的句子替换为虚线输入框：

- **绿色虚线输入框**：醒目提示需要填写的空白。
- **分隔符自动保留**：逗号、句号等标点保持在输入框之间。
- **可滚动容器**：长文章也可在固定区域内上下滚动查看。
- **最小高度**：输入框和答案显示区域均设置 `minHeight`，避免无内容时高度缩减。
