# 06 — 文本比对机制

## 文件说明

`src/utils/diff.tsx` — 基于 `diff` 库（npm: `diff`）进行字符级差异比较。

## 核心函数

### `computeDiff(original, userInput)`

```typescript
interface DiffSegment {
  text: string;
  type: 'correct' | 'incorrect' | 'missing' | 'extra';
}

function computeDiff(original: string, userInput: string): DiffSegment[]
```

使用 `diffChars()` 对原文和用户输入进行字符级 diff。返回分段列表，每段标记类型。

### 颜色标记

| 类型 | 含义 | 样式 |
|------|------|------|
| `correct` | 内容相同 | 绿色文字 `#16a34a` |
| `missing` | 原文有 → 用户漏输入 | 橙色背景 `#fed7aa` + 橙色文字 `#ea580c` |
| `extra` | 原文没有 → 用户多输入 | 红色删除线 `#dc2626` + 浅红背景 |
| `incorrect` | 替换错误（预留） | 红色背景 + 删除线 |

### 渲染组件

```tsx
<DiffView original="原文" userInput="用户输入" />
```

返回 `<span>` 元素列表，直接嵌入在文本流中。

## 判题逻辑

### 语文记忆

严格相等比对（字符串全等）：

```typescript
userInput.trim() === correct.trim()
```

### 英语记忆

忽略大小写 + 忽略首尾空白：

```typescript
userInput.trim().toLowerCase() === correct.trim().toLowerCase()
```

## 评分计算

```typescript
const score = Math.round((correctCount / total) * 100);
```

- ≥80 分：绿色显示
- 50-79 分：橙色显示
- <50 分：红色显示
