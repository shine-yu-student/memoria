# 02 — 架构设计

## 系统架构

Memoria 采用**单页应用（SPA）**架构，所有数据存储在客户端浏览器中，无需后端服务。

```

│              ├── Layout (TopBar + Sidebar + ContentCard)         │

│              │     ├── Sidebar (文章记忆/英语记忆/设置)            │
│              │     └── ContentCard (大圆角矩形)                   │
```

## 布局说明

### 全局框架

页面采用**顶栏 + 侧栏 + 内容卡片**的三段式布局：

- **顶栏 (TopBar)**：固定于页面顶部，包含品牌名和主题切换按钮。
- **侧栏 (Sidebar)**：固定于页面左侧或右侧（可设置），初始仅显示图标，鼠标悬停时展开显示图标+文字标签。底部设有设置按钮。
- **内容卡片 (ContentCard)**：位于顶栏下方、侧栏旁的大圆角矩形，所有功能页面均在其中渲染。

### 侧栏交互

- 侧栏收起时宽度 56px，展开时宽度 180px。
- 展开方式：鼠标悬停自动展开。
- 收起方式：可通过设置选择"移出鼠标"或"点击外部"。
- 内容卡片根据侧栏展开/收起状态自动调整边距。
