# 02 — 架构设计

## 系统架构

Memoria 采用**单页应用（SPA）**架构，所有数据存储在客户端浏览器中，无需后端服务。

```
┌────────────────────────────────────────────┐
│                  App.tsx                    │
│     (主应用，管理 Tab 导航)                   │
└──────────┬──────────────────────┬──────────┘
           │                      │
    ┌──────▼──────┐       ┌──────▼──────┐
    │ 语文记忆模块  │       │ 英语记忆模块  │
    │ ArticleMgr  │       │ BookManager  │
    │ MemView     │       │ EnglishMem   │
    └──────┬──────┘       └──────┬──────┘
           │                      │
    ┌──────▼──────────────────────▼──────┐
    │         通用组件层                    │
    │ Navbar / CompareResult / ImportExport│
    └──────┬──────────────────────┬──────┘
           │                      │
    ┌──────▼──────┐       ┌──────▼──────┐
    │  工具层      │       │  数据层      │
    │ splitter.ts │       │ storage.ts  │
    │ diff.tsx    │       │ (localStor) │
    │ types.ts    │       │             │
    └─────────────┘       └─────────────┘
```

## 目录结构

```
memoria/
├── index.html              # HTML 入口
├── package.json            # 依赖配置
├── vite.config.ts          # Vite 配置
├── tsconfig.json           # TypeScript 配置
├── docs/                   # 📄 文档目录
│   ├── 01-项目概览.md
│   ├── 02-架构设计.md
│   ├── 03-数据类型.md
│   ├── 04-语文记忆模块.md
│   ├── 05-英语记忆模块.md
│   ├── 06-文本比对机制.md
│   ├── 07-导入导出系统.md
│   ├── 08-持久化存储.md
│   └── 09-开发指南.md
└── src/
    ├── main.tsx             # React 入口
    ├── index.css            # 全局样式
    ├── App.tsx              # 主应用组件 + Tab 导航
    ├── vite-env.d.ts        # Vite 类型声明
    ├── types/
    │   └── index.ts         # 所有类型定义 + 工具函数
    ├── utils/
    │   ├── splitter.ts      # 中文句子切分
    │   ├── diff.tsx         # 文本比对 (React 组件)
    │   └── storage.ts       # localStorage 存储
    └── components/
        ├── common/
        │   ├── Navbar.tsx           # 顶部导航
        │   ├── CompareResult.tsx    # 记忆结果展示
        │   ├── JsonImportExport.tsx # 导入导出工具
        │   └── ImportExportPage.tsx # 导入导出页面
        ├── chinese/
        │   ├── ArticleManager.tsx   # 文章管理
        │   └── ArticleMemoryView.tsx # 语文记忆交互
        └── english/
            ├── BookManager.tsx      # 词书/句书管理
            └── EnglishMemoryView.tsx # 英语记忆交互
```

## 组件树

```
App
├── Navbar (activeTab, onTabChange)
├── [tab=chinese] ArticleManager
│   └── ArticleMemoryView
│       └── CompareResult
├── [tab=english] BookManager
│   └── EnglishMemoryView
│       └── CompareResult
└── [tab=import-export] ImportExportPage
    ├── JsonImportExport
    ├── SingleExportBtn (per article)
    ├── SingleExportBtn (per word book)
    └── SingleExportBtn (per sentence book)
```
