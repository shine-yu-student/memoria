# 01 — 项目概览

## 项目简介

**Memoria** 是一个基于 TypeScript + React 的本地化记忆助手应用。它帮助用户通过"主动回忆"的方法来记忆中文文章和英文词汇/句子。

## 功能总览

### 文章记忆模块

- **文章管理**：支持导入多篇中文文章，自动按分隔符切分为句子。文章以卡片形式展示，支持搜索、悬停删除。
- **全文编辑**：点击文章卡片进入编辑页面，可直接修改标题（点击标题）和正文内容。
- **全篇记忆**：用户凭记忆输入整篇文章，系统与原文逐字符比对并标记差异。
- **部分记忆**：
  - **随机记忆**：系统随机选取句子作为填空题，可调整选取比例。
  - **指定记忆**：用户自行点击句子选择需要记忆的内容，支持 Shift+点击范围选择，显示原文标点。
- **指导记忆**：导入 JSON 配置文件，按步骤逐步记忆文章。每步提供提示文本，支持逐步推进、回退，可仅作答部分输入框。
- **结果反馈**：以绿色（正确）、橙色（遗漏）、红色删除线（多余/错误）显示差异。

### 英语记忆模块

- **词书管理**：支持创建和管理英语单词/短语-中文对应关系的词书，以紧凑卡片形式展示单词。
- **句书管理**：支持创建和管理英语句子-中文翻译的句书。
- **词条展开编辑**：点击单词卡片展开显示中英文，可内联编辑。
- **灵活选择**：支持选择模式，通过复选框或 Shift+点击范围选择条目。
- **混合记忆**：可同时从多本书中选择条目进行记忆。
- **翻译提示**：展示中文意义，用户输入对应的英文内容。
- **计时功能**：记忆界面实时显示用时，结果页展示总用时。

### 设置系统

- **外观设置**：侧栏位置（左/右）、颜色主题（浅色/深色）、展开退出方式。
- **数据管理**：全量数据导入/导出（JSON 格式），导入时按标题合并并去重。

### 导入导出系统

- **单资源导出**：每篇文章/词书/句书可单独导出为 JSON 文件。
- **全量导出**：将所有数据导出为单个 JSON 备份文件。
- **全量导入**：从备份文件恢复全部数据（设置页数据管理中）。

## 技术栈

| 技术 | 用途 |
|------|------|
| TypeScript 5.6+ | 类型安全 |
| React 18.3+ | UI 框架 |
| Vite 6 | 构建工具 |
| diff 库 | 文本差异比较 |
| localStorage | 客户端持久化 |

## 运行方式

```bash
npm install
npm run dev      # 开发模式
npm run build    # 生产构建
npm run preview  # 预览构建结果
```
